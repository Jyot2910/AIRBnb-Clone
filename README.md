# Airbnb Landing Page Clone

A responsive Airbnb-inspired landing page built using **HTML, CSS, and Bootstrap 4**. This project recreates the Airbnb homepage hero section, featuring a navigation bar, booking form, and full-screen background image similar to Airbnb's classic landing page.

## 📸 Preview

The landing page includes:

- Airbnb-style navigation bar
- Hero background image
- Booking form with:
  - Location input
  - Check-in date
  - Check-out date
  - Guest selection dropdown
  - Search button
- Responsive design using Bootstrap

## 🚀 Features

- Fully responsive layout
- Bootstrap 4 integration
- Custom CSS styling
- Font Awesome icons
- Modern UI design
- Clean and organized code structure

## 🛠️ Technologies Used

- HTML5

## ⚙️ Setup Instructions

1. Download the project files.
2. Keep `bootstrap.html` and `custom.css` in the same folder.
3. Open `bootstrap.html` in any web browser.

No additional installation is required.
